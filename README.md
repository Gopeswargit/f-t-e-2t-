# Universal-library-docker-dencontainer# NTRCA Math & Calculus Assistant
এই প্রজেক্টটি NTRCA শিক্ষক নিয়োগ পরীক্ষার গণিত প্রস্তুতির জন্য তৈরি। এখানে ল্যাপ্লাস ট্রান্সফর্ম এবং ক্যালকুলাসের গাণিতিক সমস্যাগুলোকে সহজ লজিক ও MCQ এর মাধ্যমে সমাধান করার ব্যবস্থা রাখা হয়েছে।

## কেন এটি আলাদা?
- পরীক্ষার উপযোগী শর্টকাট টেকনিক।
- ল্যাপ্লাস ও ক্যালকুলাসের সিলেবাস-ভিত্তিক গাণিতিক সমাধান।
- MCQ প্র্যাকটিস করার জন্য দ্রুত গাণিতিক লজিক।

## ব্যবহারবিধি
আপনার কম্পিউটারে পাইথন ইন্সটল থাকলে নিচের কমান্ডটি রান করুন:
python math_solver.py
